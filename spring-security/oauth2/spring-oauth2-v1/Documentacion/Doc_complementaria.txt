Desde servidor de autorización, entrar a este endpoint para obtener información de endpoints de oauth2:
http://localhost:9000/.well-known/openid-configuration

Pagina para cliente de pruebas:
https://oauthdebugger.com/

Pagina para verificar info del token devuelto por servidor de autorización:
https://jwt.io/
